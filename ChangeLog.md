# ChangeLog

## v0.1.0 - 第一版本

### 新增

- 新增 Chrome Manifest V3 扩展形态，支持浏览器工具栏 popup 和完整管理面板。
- 新增项目分组、项目管理、字段模板、账号管理和系统设置模块。
- 新增按当前页面地址匹配环境，并在匹配环境中选择账号填充的能力。
- 新增敏感字段加密存储和通行密钥验证，查看敏感值与导出明文前需要解锁。
- 新增 JSON 导入、导出能力，便于备份和迁移账号配置。
- 新增 MinIO 同步配置，支持本地配置与远程对象存储之间同步。

### 修改

- 优化左侧菜单折叠/展开动画，并接入独立菜单图标资源。
- 优化扩展 popup 中匹配环境、账号列表和账号摘要的展示方式。
- 优化字段配置弹窗输入框、下拉框、表格和勾选控件的视觉样式。
- 调整扩展工具栏图标逻辑，改为根据当前页面是否匹配可用环境显示状态。
